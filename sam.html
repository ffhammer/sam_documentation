    SAM Settings¶

    class sam.SAM_Settings(normalize_survival_for_stress_conversion=False, stress_form='div', transform=<function transform_williams_and_linear_interpolation>, beta_q=3.2, beta_p=3.2, curve_fit_lib='scipy', fix_f_parameter_ll5=None, keep_co_stressor_f_parameter_free=False, e_param_fac=1.0)¶

    Bases: object

    Settings for configuring the SAM (Stress Addition Model) used in concentration-response predictions.

    normalize_survival_for_stress_conversion: bool = False¶

    If True, normalizes survival values based on environmental stress.

    stress_form: str = 'div'¶

    Defines the formula used to calculate environmental stress. Supported options: - “div”: additional_stress = stressor / control - “subtract”: additional_stress = 1 - (control - stressor) - “only_stress”: additional_stress = 1 - stressor - “stress_sub”: Subtracts transformed control stress from transformed stressor stress.

    transform(surv)¶

    Transformation function applied to regression data prior to model fitting.

    beta_q: float = 3.2¶

    q Parameter of beta distribution for survival to stress and vice versa conversions

    beta_p: float = 3.2¶

    p Parameter of beta distribution for survival to stress and vice versa conversions

    curve_fit_lib: str = 'scipy'¶

    //github.com/MockaWolke/py_lmcurve_ll5

    Type:

    Controls which library is used for DoseResponse Curve fitting. Either scipy for scipy.optimize.curve_fit or lmcurce for using https

    sam.generate_sam_prediction(concentration, control_survival, co_stressor_survival, max_survival=None, settings=SAM_Settings(normalize_survival_for_stress_conversion=True, stress_form='div', transform=<function transform_williams_and_linear_interpolation>, beta_q=3.2, beta_p=3.2, curve_fit_lib='scipy', fix_f_parameter_ll5=1.0, keep_co_stressor_f_parameter_free=False, e_param_fac=1.0), control_name='Control', stressor_name='Stressor')¶

    Computes survival and stress predictions based on a main control series and a stressor series using the Stress Addition Model (SAM).

    Parameters:

    • control_data (CauseEffectData) – Concentration-response data for the control group.

    • co_stressor_data (CauseEffectData) – Concentration-response data for the stressor condition.

    • meta (Optional[ExperimentMetaData]) – Metadata, used to infer max survival if not provided.

    • max_survival (Optional[float]) – Maximum survival rate. Overrides meta.max_survival if given.

    • settings (SAM_Settings) – Configuration settings for SAM. Controls stress computation formula, normalization, and additional adjustments.

    Return type:

    SAMPrediction

    Returns:

    SAMPrediction

    Example

    `python prediction =  SAMPrediction.generate(control_data, co_stressor_data, settings=SAM_Settings(stress_form="div")) `
